# YouTube Clone

A React-based YouTube clone application built with Vite.

## Live Demo
Check out the live application here: [YouTube Clone Demo](https://youtube-clone-rosy-gamma-79.vercel.app/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/jacob4521/youtube-clone.git
   ```
2. Navigate to the project directory
   ```sh
   cd youtube-clone
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Get a free API Key at [Google Cloud Console](https://console.cloud.google.com/) for the YouTube Data API v3.
5. Create a `.env` file in the root directory and enter your API key:
   ```env
   VITE_YOUTUBE_API_KEY=YOUR_API_KEY_HERE
   ```
6. Start the development server
   ```sh
   npm run dev
   ```
